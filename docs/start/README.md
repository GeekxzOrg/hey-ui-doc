### 快速上手

Hey-ui 一个基于css的高质量UI组件库.

> 在开始之前，推荐先学习微信官方的小程序开发文档，并正确安装和配置了 微信开发者工具。 同时，我们假设你已了解关于 HTML、CSS 和 JavaScript 的初级知识，并且已经熟悉并阅读了微信小程序文档


### 安装

直接通过git下载 Hey UI 源代码，并将 dist 目录（Hey-UI 文件夹）拷贝到自己的项目中。

```sh
git clone https://github.com/GeekxzOrg/Hey-UI.git

```
