### 线条 Line

### 线条颜色

颜色有主题配置class
|  	class   		|	说明	|
|  	:----  			| 	:----  	|
|	h-line-{color}	|	color可选值:主题配置文字颜色值



### 线条大小与形状

线条大小与形状配置class
|  	class   		|	说明	|
|  	:----  			| 	:----  	|
|	h-line-{type}-line1	|	type可选值:slide或者dashed
|	h-line-{type}-line2	|	type可选值:slide或者dashed
|	h-line-{type}-line3	|	type可选值:slide或者dashed
